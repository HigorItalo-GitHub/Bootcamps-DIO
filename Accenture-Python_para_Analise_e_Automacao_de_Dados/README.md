<h3>💡 Insights Obtidos</h3>



<ul>

&#x20; <li>Prompts genéricos geram respostas superficiais</li>

&#x20; <li>Definir contexto melhora drasticamente os resultados</li>

&#x20; <li>Iteração é essencial para obter respostas de qualidade</li>

</ul>

<h3>🧠 Engenharia de Prompts</h3>



<h4>Prompt Inicial</h4>

<pre>

Explique como usar pandas para análise de dados.

</pre>



<p><b>Resultado:</b> resposta genérica e pouco útil.</p>



<h4>Prompt Refinado</h4>

<pre>

Explique como usar pandas com exemplos de leitura, limpeza e agregação.

</pre>



<p><b>Resultado:</b> melhora na clareza e exemplos.</p>



<h4>Prompt Estratégico</h4>

<pre>

Você é um engenheiro de dados. Explique um pipeline real usando pandas.

</pre>



<p><b>Resultado:</b> resposta estruturada e aplicável.</p>
