<h2>🧠 3. Engenharia de Prompts e "Cicatrizes"</h2>

<h3>🔍 Prompt Inicial</h3>
<pre>
Explique como usar pandas para análise de dados.
</pre>

<p><b>Problema:</b> resposta genérica e pouco aplicável.</p>

<h3>🔁 Refinamento</h3>
<pre>
Explique como usar pandas com exemplos práticos de leitura, limpeza e agregação.
</pre>

<p><b>Resultado:</b> melhora na estrutura e exemplos.</p>

<h3>🎯 Prompt Estratégico</h3>
<pre>
Você é um engenheiro de dados. Explique um pipeline real usando pandas.
</pre>

<p><b>Resultado:</b> resposta estruturada, contextualizada e prática.</p>

<h3>⚠️ Dificuldades Encontradas</h3>

<ul>
  <li>Respostas superficiais com prompts genéricos</li>
  <li>Necessidade de contexto explícito</li>
  <li>Omissão de erros comuns</li>
  <li>Refinamento iterativo necessário</li>
</ul>

<h3>🛠️ Estratégias que Funcionaram</h3>

<ul>
  <li>Definir papel da IA</li>
  <li>Solicitar exemplos práticos</li>
  <li>Exigir estrutura passo a passo</li>
  <li>Incluir erros comuns e boas práticas</li>
</ul>
