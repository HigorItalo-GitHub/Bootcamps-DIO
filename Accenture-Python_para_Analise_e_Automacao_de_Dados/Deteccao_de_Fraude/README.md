<h1 align="center">💳 Detecção de Fraudes em Cartão de Crédito</h1>

<p align="center">
Machine Learning aplicado à identificação de transações fraudulentas.
</p>

<hr>

<h2>📌 Sobre o Projeto</h2>

<p>
Projeto desenvolvido para detectar fraudes em transações de cartão de crédito utilizando técnicas modernas de Machine Learning.
</p>

<ul>
<li>Regressão Logística</li>
<li>Random Forest</li>
<li>XGBoost</li>
<li>SHAP Explainability</li>
</ul>

<hr>

<h2>📂 Dataset</h2>

<p>
Base pública disponibilizada pelo TensorFlow.
</p>

<p>
284.807 transações<br>
492 fraudes detectadas
</p>

<hr>

<h2>📈 Métricas Utilizadas</h2>

<li>Recall</li>

<li>ROC AUC</li>
<li>Precision-Recall Curve</li>
</ul>
